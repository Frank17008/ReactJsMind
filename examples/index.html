ReactJsMind 思维导图组件

GitHub npm jsMind
主题: primary
  • 点击节点 - 选中节点，查看节点信息
  • 双击节点 - 编辑节点内容
  • 拖拽节点 - 移动节点位置（编辑模式下）
  • 展开/收起 - 点击节点旁的 + / - 符号
  • 鼠标悬停 - 高亮显示当前节点
  • Ctrl+ 滚轮缩放 - 放大或缩小思维导图
  • Space - 选中节点,Space空格键可以展开/收起节点
  • Insert - 添加子节点
  • Enter - 添加兄弟节点
  • F2 - 编辑当前节点
  • Delete - 删除节点
  • Space - 展开/收起节点
  • ← → ↑ ↓ - 导航节点
  • Ctrl + A - 全选节点
  • Ctrl + Z - 撤销操作
  • 支持树形和数组两种数据格式
  • 支持节点拖拽和缩放
  • 支持自定义节点样式
  • 支持截图导出功能
  • 支持主题切换
  • 支持编辑模式切换
  • 支持多种布局方向
  • 支持节点折叠/展开
